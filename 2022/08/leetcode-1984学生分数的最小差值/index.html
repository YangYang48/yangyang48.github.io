LeetCode 1984学生分数的最小差值

给你一个 下标从 0 开始 的整数数组 nums ，其中 nums[i] 表示第 i 名学生的分数。另给你一个整数 k 。

从数组中选出任意 k 名学生的分数，使这 k 个分数间 最高分 和 最低分 的 差值 达到 最小化 。

返回可能的 最小差值 。

示例 1：

1输入：nums = [90], k = 1
2输出：0
3解释：选出 1 名学生的分数，仅有 1 种方法：
4- [90] 最高分和最低分之间的差值是 90 - 90 = 0
5可能的最小差值是 0

示例 2：

 1输入：nums = [9,4,1,7], k = 2
 2输出：2
 3解释：选出 2 名学生的分数，有 6 种方法：
 4- [9,4,1,7] 最高分和最低分之间的差值是 9 - 4 = 5
 5- [9,4,1,7] 最高分和最低分之间的差值是 9 - 1 = 8
 6- [9,4,1,7] 最高分和最低分之间的差值是 9 - 7 = 2
 7- [9,4,1,7] 最高分和最低分之间的差值是 4 - 1 = 3
 8- [9,4,1,7] 最高分和最低分之间的差值是 7 - 4 = 3
 9- [9,4,1,7] 最高分和最低分之间的差值是 7 - 1 = 6
10可能的最小差值是 2

提示：

  • 1 <= k <= nums.length <= 1000
  • 0 <= nums[i] <= 105

思路

进行排序+滑动窗口

首先将无序数组进行从小到大排序

开始遍历数组，使用一个大小固定为 k的滑动窗口在 nums上进行遍历，那么窗口中k名学生的最高分和最低分的差值在任意一次都是固定值，找到这个最小值即可

解题

 1class Solution {
 2    //排序加滑动窗口
 3    public int minimumDifference(int[] nums, int k) {
 4        int len = nums.length;
 5        int min = Integer.MAX_VALUE;
 6        Arrays.sort(nums);
 7        for(int i = 0; i + k -1 < len; i++)
 8        {
 9            min = Math.min(min, nums[i + k - 1] - nums[i]);
10        }
11        return min;
12    }
13}
